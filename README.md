A simple checkout payment confirmation component.

## Getting Started

```bash
# git clone repo
git clone https://github.com/mukolweke/NuxtCheckoutForm.git

# enter the project folder
cd NuxtCheckoutForm

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
